1. 미디어 쿼리
    - 미디어 쿼리란 CSS3 모듈 중 하나
    - 접속하는 사용자의 장치에 따라 특정한 stylesheet를 적용하는 기술
    - @media 키워드로 지정
    - @media screen and(min-width:px값) and (max-width:px값) {
        최소 화면 크기 ~ 최대 화면 크기에서 적용될 스타일 시트 작성
    }
    - 반응형 웹사이트를 만드는 데 적용하는 기술
    - 반응형 웹사이트란 접속한 기기별로 레이아웃을 변경해주는 기술(스타벅스 홈페이지 잘돼있음)
    - 미디어 쿼리를 적용하기 위해서 html 문서의 head 부분에 meta 태그로
      viewport 지정해야 됨

2. 플렉서블 박스
    - 플렉서블 박스란 그리드 레이아웃을 기본으로 요소들을 원하는 위치에 배치하는 기술
    - 화면 크기의 여유 공간에 따라 위치나 크기 또는 높이를 자유롭게 조절 할 수 있다.
    - 기본적으로 플렉서블 박스로 지정할 요소를 display : flex;로 지정
      그 안의 요소들을 flex정렬 기능을 이용해 정렬 시킨다.
      1) flex-direction 속성 : 정렬의 기준이 되는 주축 정하기
                               기본은 왼쪽에서 오른쪽 방향으로 수평
      2) flex-wrap 속성 : 플렉스 요소들을 배치하는 방법 정하기
                          no-wrap : 기본 값. 한 줄로 배치
                          wrap : 여러 줄로 배치
                          wrap-reverse : 여러 줄로 배치 하되 주축과 반대 방향으로 배치
      3) order 속성 : 플렉스 요소들의 배치 순서 정하기
                      기본적으로 먼저 배치 한 요소들이 앞에 위치하지만 
                      order 속성을 이용하면 요소들의 배치 순서를 바꿀 수 있다.
      4) justify-content 속성 : 주축 기준으로 플렉스 요소들의 배치 방법 지정하기
                                flex-start : 주축을 시작점을 기준으로 배치
                                flex-end : 주축의 끝점을 기준으로 배치
                                center : 주축의 중앙을 기준으로 배치
                                space-between : 첫 번째 요소는 주축의 시작점에,
                                                마지막 요소는 주축의 끝점에 배치하고
                                                나머지 요소들은 가운데 배치하되 동일한 간격을 가지고 배치
      5) space-around : 모든 요소들을 같은 간격으로 배치
